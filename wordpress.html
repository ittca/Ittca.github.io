Resenha wordpress

Estrutura e comandos essenciais

Para um tema wordpress existem dois ficheiros essenciais:

index.php
style.css

O index pode conter apenas uma estrutura html e o style a decoração que pretendermos, no entanto vários são outros os ficheiros que vão permitir que a plataforma wordpress possa introduzir e mudar dados ao nosso tema, e o wordpress também têm as suas próprias funções para estas questões. As Linguagens permitidas são html, css, javascript e php.
O Style.css deverá começar com os comentários:
/*
Theme Name: resenha wordpress
Author: Tiago Calado
Version: 1.0
*/
vair dar ao tema os dados mencionados, e o wordpress vai lê-los e disponibilizá-los no lugar apropriado.

header.php
footer.php

Ao usar estes dois ficheiros vamos iniciar a estrutura html no header.php e terminá-la no footer.php, depois no index.php usa-se código php e chama-se no ínicio a get_header() e no fim get_footer().
